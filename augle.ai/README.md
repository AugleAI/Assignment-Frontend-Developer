# How to Run this App locally

## `clone this repos`

```
$ git clone https://github.com/mrmani15/Assignment-Frontend-Developer.git
```

## `Go inside Folder`

```
$ cd Assignment-Frontend-Developer/augle.ai
```

## `Installing dependancies`

```
npm install
```

## `npm start`

```
npm start
```

### `Open Browser`

```
http://localhost:3000
```
